# Finance App

Aplikasi pelacak keuangan (Finance Tracker) sederhana yang dibangun menggunakan [Flutter](https://flutter.dev/). Aplikasi ini memiliki fitur login, dashboard, pencatatan transaksi (pemasukan & pengeluaran), serta antarmuka bergaya komik yang unik dan interaktif.

## Fitur Utama
- **Dashboard**: Ringkasan Saldo, Pemasukan, dan Pengeluaran secara *real-time*.
- **Manajemen Transaksi**: Tambah, ubah, dan hapus histori transaksi.
- **Tema (Theme)**: Mendukung Mode Gelap (Dark Mode) dan Mode Terang (Light Mode).
- **Kategori**: Pilih ikon kategori untuk setiap transaksi (Makanan, Transportasi, Gaji, dll).

## Cara Menjalankan Proyek (How to Run)

Ikuti petunjuk di bawah ini untuk mengkloning dan menjalankan aplikasi ini di komputer lokal Anda.

### Prasyarat (Prerequisites)
Pastikan Anda sudah menginstal beberapa aplikasi dasar berikut:
1. **[Flutter SDK](https://docs.flutter.dev/get-started/install)** (termasuk Dart)
2. **IDE / Code Editor**: [VS Code](https://code.visualstudio.com/) (disarankan dengan ekstensi Flutter) atau [Android Studio](https://developer.android.com/studio).
3. Emulator Android, Simulator iOS, web browser (Chrome), atau perangkat ponsel pintar (device) yang disambungkan menggunakan kabel USB.

### Langkah-langkah (Steps)

1. **Kloning Repository**
   Buka terminal, Command Prompt, atau PowerShell Anda lalu jalankan perintah berikut:
   ```bash
   git clone https://github.com/anadriannugraha/finance_app.git
   cd finance_app
   ```

2. **Unduh Dependencies (Packages)**
   Ambil semua paket (library) pendukung yang dibutuhkan oleh proyek ini:
   ```bash
   flutter pub get
   ```

3. **Jalankan Aplikasi**
   Setelah semuanya siap, Anda bisa langsung menjalankan aplikasi menggunakan perintah:
   ```bash
   flutter run
   ```
   *(Atau Anda bisa membuka folder `finance_app` di VS Code / Android Studio dan cukup menekan tombol `Run` atau `F5`)*.
